# Этап 1: Установка pfSense

## 1.1 Создание виртуальной машины

### Параметры ВМ:
- **RAM:** 2048 MB
- **HDD:** 10 GB (VDI, динамический)
- **Сетевые адаптеры:** 2

### Настройка сети:
- **Adapter 1 (WAN):** NAT
- **Adapter 2 (LAN):** Internal Network "lan"

## 1.2 Установка pfSense

1. Загрузился с ISO образа pfSense 2.8.1
2. Выбрал "Install"
3. Принял настройки по умолчанию
4. Назначил интерфейсы:
   - WAN: em0
   - LAN: em1
5. Установил IP для LAN: 192.168.1.1/24

## 1.3 Первоначальная настройка

После установки и перезагрузки:
- WAN получил IP по DHCP: 10.0.2.15/24
- LAN имеет статический IP: 192.168.1.1/24
- Доступен консольный интерфейс с 16 опциями
